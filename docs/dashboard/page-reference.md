# Page reference
Home, Market Intelligence, Scanner, COA Research, Strategy Lab, Research Knowledge, Portfolio, Options Analytics, Trade Journal, Performance, Execution, Operations Center, Alerts, and Configuration are available through the shared navigation.

## Configuration

Configuration is a local operational console. It displays only masked credential presence, never raw Dhan, Fyers, or Telegram values. Local saves use the operating-system credential manager; deployed applications read environment variables or Streamlit secrets. Runtime settings and version history are stored outside the repository in the user's CQRP configuration directory.

Execution mode is limited to `DISABLED` and `PAPER`. The page cannot enable live order placement.
