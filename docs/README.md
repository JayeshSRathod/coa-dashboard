# CQRP documentation

This directory is the version-controlled source of product, architecture, research, and operational decisions for the COA Quant Research Platform (CQRP).

- `adr/`: architecture decision records
- `architecture/`: system boundaries and module contracts
- `development/`: contributor and coding standards
- `sprints/`: approved sprint scopes and acceptance criteria

The existing COA logic is the v1 baseline. CQRP v2 evolves it through additive, replayable research infrastructure.
